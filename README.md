# 3. Duplicate-Image Blocking

## Why does Kitab AI block duplicate images?
To save space and improve clarity, Kitab AI uses:
- pHash (perceptual hashing)  
- Image fingerprinting  
- Similarity scoring

If an image is identical or almost identical to a previous upload, it is rejected.

## Can I bypass this?
Duplicate detection cannot be disabled, but you can:
- Crop the image  
- Resize it  
- Upload a different version

# 7. Troubleshooting

## Upload failures
Check:
- File size  
- Internet connection  
- Duplicate-image flag  
- Unsupported formats

## OCR issues
Improve text detection by:
- Uploading sharper scans  
- Avoiding blurry or dark images  
- Using 300 DPI or higher when scanning

## App stuck or crashing?
Try:
1. Force close the app  
2. Reopen the document  
3. Clear cache  
4. Reinstall the app (your data stays safe)
